### <strong>Feel free to use these</strong>
{.text-h6}

You don't need to ask permission to use the badges to this site. Just grab them, hotlink them, display them on your website. 

### <strong>Keep the badge's pixelated style</strong>
{.text-h6}

If you're using the badge, please use image-rendering: pixelated (or crisp-edges for Firefox) in your CSS. Without it, browsers will blur the pixels and it'll look like garbage. It's pixel art, please treat it like pixel art.

### <strong>Don't be weird about it</strong>
{.text-h6}

Don't use these to impersonate me, claim I endorsed something I didn't, or make it look like I'm affiliated with something sketchy. I trust you know the difference between linking to a site you like and pretending to be someone else. If you're unsure whether what you're doing is weird, it probably is.
